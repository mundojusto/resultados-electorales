# Datos oficiales

Esta carpeta contiene los **datos oficiales** de los resultados electorales de M+J.

Sube aquí los archivos tal como los publica la fuente oficial.

## Recomendaciones

- **Formatos preferidos:** `.csv` o `.json` (también se aceptan `.xlsx`, `.pdf`
  u otros, pero los formatos abiertos facilitan su uso en la web/app).
- **Nombres de archivo:** usa nombres descriptivos, en minúsculas y sin espacios.
  Ejemplos:
  - `2026-resultados-generales.csv`
  - `2026-resultados-por-mesa.json`
- **No modifiques los datos originales.** Si necesitas una versión procesada,
  guárdala con un sufijo claro (p. ej. `*-procesado.csv`).
- Incluye, cuando sea posible, la **fuente** y la **fecha de descarga** de cada
  archivo en este README o en un archivo `FUENTES.md`.
